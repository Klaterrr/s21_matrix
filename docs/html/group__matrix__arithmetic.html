◆ s21_mult_number()

int s21_mult_number ( matrix_t * A,
double number,
matrix_t * result )

Умножает матрицу на скалярное значение.

Умножает каждый элемент матрицы A на число number и сохраняет результат в result. Функция сама создает результирующую матрицу result.

Аргументы
[in]AУказатель на матрицу для умножения.
[in]numberСкалярное значение, на которое умножается матрица.
[out]resultУказатель на результирующую матрицу (произведение).
Возвращает
Код результата операции:
OK (0) - в случае успеха.
INCORRECT_MATRIX (1) - если матрица A некорректна или не удалось создать результирующую матрицу.

◆ s21_sub_matrix()

int s21_sub_matrix ( matrix_t * A,
matrix_t * B,
matrix_t * result )

Вычитает одну матрицу из другой.

Вычитает матрицу B из матрицы A и сохраняет результат в result. Матрицы A и B должны иметь одинаковые размеры. Функция сама создает результирующую матрицу result.

Аргументы
[in]AУказатель на уменьшаемое.
[in]BУказатель на вычитаемое.
[out]resultУказатель на результирующую матрицу (разность).
Возвращает
Код результата операции:
OK (0) - в случае успеха.
INCORRECT_MATRIX (1) - если одна из матриц A или B некорректна.
CALCULATION_ERROR (2) - если размеры матриц A и B не совпадают.

◆ s21_inverse_matrix()

int s21_inverse_matrix ( matrix_t * A,
matrix_t * result )

Находит обратную матрицу.

Обратная матрица существует только для квадратных матриц с ненулевым определителем.

Аргументы
[in]AУказатель на исходную матрицу.
[out]resultУказатель на результирующую обратную матрицу.
Возвращает
OK (0) - успех.
INCORRECT_MATRIX (1) - если матрица некорректна.
CALCULATION_ERROR (2) - если матрица не квадратная или ее определитель равен 0.

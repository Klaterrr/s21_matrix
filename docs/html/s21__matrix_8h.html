Подробное описание

Заголовочный файл для библиотеки работы с матрицами.

Автор
Johan Kamiyama (a.k.a @kuretaru or shanaecl)

Содержит объявление структуры matrix_t и прототипы всех публичных функций для создания, удаления, сравнения матриц, а также для выполнения арифметических и алгебраических операций.

END_TEST Suite * s21_matrix_suite ( void )

Создает и конфигурирует тестовый набор (Suite).

Объединяет все тестовые случаи (TCase), сгруппированные по функциональности (основная, обработка ошибок), в один набор для последующего запуска через SRunner.
